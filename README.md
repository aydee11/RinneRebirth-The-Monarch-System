# RINNE REBIRTH
## System Reset Protocol

A personal development app designed to help you reset and refocus your goals and tasks.

### Features

- **Focus Tasks**: Track and manage your priority tasks with a clean, minimal interface
- **Sync Status**: Real-time sync status monitoring
- **Dark Theme**: Easy on the eyes with a modern dark interface powered by Tailwind CSS
- **Glass Morphism**: Beautiful glassmorphic design elements with violet/fuchsia gradient accents

### Getting Started

Simply open `index.html` in your browser to launch the app.

### Design

- Built with vanilla HTML/CSS/JavaScript
- Styled with Tailwind CSS
- Features a dark theme (#030303) with gradient text and glassmorphic components
